# Flowly — Concept A

## Статус

- **Источник:** предоставлен пользователем 2026-07-13 как исходный вариант HTML-мокапа.
- **Роль:** референс визуального направления и частично интерактивный screen concept.
- **Approval:** не зафиксирован; концепт не считается финальным дизайном и не закрывает E0-D0-T01–T06.
- **Исходный каталог:** `flowly_html_mockup/` в корне проекта.
- **Текущее расположение:** `docs/design/screens/concept-a/`.

## Содержимое

- `index.html` — основная версия с внешними assets.
- `flowly-mockup-standalone.html` — переносимая standalone-версия.
- `assets/flowly-logo.svg` — полный логотип Flowly.
- `assets/flowly-icon.svg` — иконка приложения.
- `assets/*.jpg` — изображения концепта.
- `preview.png` — browser snapshot основной версии.
- `README.txt` — исходная инструкция запуска.

## Фактическое покрытие

Концепт показывает четыре mobile-first экрана:

1. Главная: дневной прогресс, текущая программа, привычки, CTA тренировки.
2. Каталог тренировок: категории, фильтры и карточки.
3. Мой ритм: дневной прогресс привычек и добавление привычки.
4. Календарь: месяц и сводка выбранного дня.

Также представлены визуальное направление, палитра, типографика, набор иконок и логотип.

Доступные демонстрационные взаимодействия:

- toast для действий и фильтров;
- переключение выполнения привычек и пересчёт прогресса;
- выбор активного пункта локальной нижней навигации;
- выбор дня календаря.

## Пробелы относительно PRD и этапа 0

- Нет пятой основной вкладки «Программы» из PRD §9.
- Нет дополнительных разделов профиля: друзья, челленджи, избранное, отчёты, настройки, уведомления, экспорт и удаление данных.
- Не покрыты онбординг, Telegram/fallback-состояния и большинство функциональных flows PRD §10–39.
- Навигация визуально переключает active state, но не связывает четыре экрана в единый user flow.
- Не представлены обязательные loading, empty, error, retry, offline и saved states PRD §40.2.
- Не показана тёмная тема, safe-area matrix, responsive-проверка 360–430 px и широких экранов.
- Нет подтверждённой accessibility-проверки по PRD §40.3.
- Это не полный интерактивный прототип E0-D0-T05 и не утверждённый пакет E0-D0-T06.

## Локальный просмотр

```bash
cd docs/design/screens/concept-a
python3 -m http.server 8000
```

Открыть `http://localhost:8000/`. Для просмотра без локальных assets можно открыть `flowly-mockup-standalone.html`.

## Зафиксированная проверка импорта

- `index.html` открыт через локальный HTTP server.
- Основные assets загружаются; единственная browser-console ошибка — отсутствующий необязательный `favicon.ico`.
- Browser snapshot сохранён как `preview.png`.
- Исходные HTML, SVG, JPG и `README.txt` перенесены без функциональных изменений; `.DS_Store` исключён как системный мусор.
